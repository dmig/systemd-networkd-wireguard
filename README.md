# systemd-networkd-wireguard

`systemd-networkd` Wireguard configurator

## Use cases:

1. Server confifguration
   1. Create server keys
   2. Set network subnets
   3. set listen port
   4. set interface name and description
2. Peer actions
   1. create peer
   2. set allowedips
   3. delete peer
   4. export peer config
